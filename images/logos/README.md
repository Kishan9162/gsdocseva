# Official Exam Logos

Place official exam logos in this folder. Logo files should be PNG format with transparent backgrounds.

## Required Logo Files:

1. **ssc.png** - Staff Selection Commission logo
   - Get from: https://ssc.nic.in/
   
2. **bihar-ssc.png** - Bihar SSC logo
   - Get from: https://bssc.bihar.gov.in/

3. **railway.png** - Indian Railways logo
   - Get from: https://rrbcdg.gov.in/

4. **upsc.png** - UPSC (Union Public Service Commission) logo
   - Get from: https://upsc.gov.in/

5. **bpsc.png** - BPSC (Bihar Public Service Commission) logo
   - Get from: https://bpsc.bih.nic.in/

6. **ibps.png** - IBPS Banking logo
   - Get from: https://ibps.in/

7. **police.png** - Police recruitment logo
   - Use state-specific police logos

8. **tnpsc.png** - Tamil Nadu PSC logo
   - Get from: https://tnpsc.gov.in/

9. **kvs.png** - Kendriya Vidyalaya Sangathan logo
   - Get from: https://kvsangathan.nic.in/

## Logo Specifications:
- **Format:** PNG with transparent background
- **Size:** 200x200px to 400x400px
- **Quality:** High resolution, clear logos
- **Naming:** Use lowercase with hyphens (e.g., bihar-ssc.png)

## How to Add:
1. Download official logo from exam authority website
2. Remove background (use remove.bg if needed)
3. Resize to 300x300px
4. Save as PNG in this folder
5. Name it exactly as shown above

## Fallback:
If logo image is not found, emoji icon will be displayed automatically.
